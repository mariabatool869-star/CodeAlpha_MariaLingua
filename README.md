# 🌍 MariaLingua — Maria's Global Translator

A beautiful, single-file web translation tool supporting **100+ languages**.

## ✨ Features

- Searchable language dropdowns (From / To)
- Auto-detect source language
- Translate via MyMemory API (with LibreTranslate fallback)
- Copy translation, text-to-speech, swap languages
- Saves your last language pair in the browser
- Responsive design — works on mobile and desktop

## 🚀 How to use

1. Clone or download this repo
2. Open `index.html` in your browser
3. Type text, choose languages, and click **Translate**

**Keyboard shortcut:** `Ctrl + Enter` to translate

## 🎨 Built with

- HTML, CSS, JavaScript (single file)
- MyMemory Translation API
- Web Speech API for text-to-speech
